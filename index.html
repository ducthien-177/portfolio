LED制御のWEBシステム

システムの全体像

このプロジェクトはRaspberry Pi（ラズパイ）・サーバーとしてFIREBASEを使用しました．開発言語はPythonです．

Part 1
温度のセンサーを定期的にFireStoreに送信し，このデータでグラーフを制作します.
Part 2
index.htmlからLEDの状態を変え，状態のデータ（ON, OFF）もFireStoreに保存されます．この状態により，ラスパイでLEDを制御します．

ソースコードは github

参考書検索システム

AIの力をWebに応用したいので，このフロジェットを開発しました．
Teachable Machine はGoogleが提供している無料のwebサービスです。 このwebサービスは、機械学習コードを作成せずに、画像、音声、ポーズを認識するようコンピュータをトレーニングでき、作成したモデルは、独自のプロジェクト（サイト、アプリなど）で使用可能です。
Teachable Machineにより，高専4年生の教科書から，高専の図書館の関連参考書を検索するシステムである．
